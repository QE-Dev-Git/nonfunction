# ISO/IEC 25010 品質特性と品質副特性

このリポジトリは、ISO/IEC 25010 の品質モデルに基づき、
**品質特性（Characteristics）** と **品質副特性（Sub-characteristics）** の関係を整理したものです。

## 構造のルール

- 品質特性 = 上位概念
- 品質副特性 = 各品質特性に属する下位概念
- 各品質副特性は個別の Markdown ファイルで詳細を記述する

## 品質特性と品質副特性の対応

### 1. 機能適合性（Functional Suitability）
ソフトウェアが、利用目的に合った機能を提供できる度合い。

この品質特性に属する品質副特性:
- [機能完全性](./機能完全性.md)
- [機能正確性](./機能正確性.md)
- [機能適切性](./機能適切性.md)

### 2. 性能効率性（Performance Efficiency）
要求された性能を、使用する資源量に対して効率よく発揮できる度合い。

この品質特性に属する品質副特性:
- [時間効率性](./時間効率性.md)
- [資源効率性](./資源効率性.md)
- [容量満足性](./容量満足性.md)

### 3. 互換性（Compatibility）
他の製品・システム・環境と共存または連携できる度合い。

この品質特性に属する品質副特性:
- [共存性](./共存性.md)
- [相互運用性](./相互運用性.md)

### 4. 使用性（Usability）
指定された利用者が、指定された利用状況において、有効性・効率性・満足性をもって利用できる度合い。

この品質特性に属する品質副特性:
- [適切性認識性](./適切性認識性.md)
- [習得性](./習得性.md)
- [操作性](./操作性.md)
- [ユーザーエラー防止性](./ユーザーエラー防止性.md)
- [ユーザーインターフェース美感性](./ユーザーインターフェース美感性.md)
- [アクセシビリティ](./アクセシビリティ.md)

### 5. 信頼性（Reliability）
指定された条件下で、指定された期間、システムやソフトウェアが機能を維持できる度合い。

この品質特性に属する品質副特性:
- [成熟性](./成熟性.md)
- [可用性](./可用性.md)
- [障害許容性](./障害許容性.md)
- [回復性](./回復性.md)

### 6. セキュリティ（Security）
情報やデータを保護し、正当な主体だけが適切にアクセスできる度合い。

この品質特性に属する品質副特性:
- [機密性](./機密性.md)
- [完全性](./完全性.md)
- [否認防止性](./否認防止性.md)
- [責任追跡性](./責任追跡性.md)
- [真正性](./真正性.md)

### 7. 保守性（Maintainability）
修正、改善、解析、再利用、試験などを効率よく行える度合い。

この品質特性に属する品質副特性:
- [モジュール性](./モジュール性.md)
- [再利用性](./再利用性.md)
- [解析性](./解析性.md)
- [修正性](./修正性.md)
- [試験性](./試験性.md)

### 8. 移植性（Portability）
他の環境へ移行、設置、置換しやすい度合い。

この品質特性に属する品質副特性:
- [適応性](./適応性.md)
- [設置性](./設置性.md)
- [置換性](./置換性.md)
